# Student Management System (Java + MySQL)

A console-based application to manage student records using Java, JDBC, and MySQL.  
This project performs CRUD operations: Add, View, Update, and Delete students.

## Technologies Used
- Java
- MySQL
- JDBC
- VS Code

## Features
- Add student records
- View all student records
- Update student details
- Delete student records

## How to Run
1. Import the project into VS Code
2. Add MySQL Connector JAR to classpath
3. Update DB username and password in `connect()` method
4. Run `StudentManagement.java`
